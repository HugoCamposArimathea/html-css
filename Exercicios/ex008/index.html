Principais formatações

Negrito / Destaque

Nessa frase temos uma frase em negrito usando a tag <b>(Não semântica)

Nesta temos um termo em destaque usando a tag <strong>

Itálico ou ênfase

Nesta frase temos um termo em itálico usando a tag <i>(não semântica

Nesta frase temos um termo em destaque usando a tag <em>

Texto marcado

Podemos criar também um texto marcado , usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande, e um texto pequeno usando a tag <small>, neste parágrafo

Texto deletado

Podemos marcar um texto como excluído para para indicar que ele deve ser lido, mas não considerado. Adicionando a tag <del>

Texto inserido

Podemos colocar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. adicionando a tag <ins>

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O usando a tag <sub>
